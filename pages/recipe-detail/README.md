# 菜谱详情页

## 页面功能

### 📱 页面布局
1. **顶部主图** - 展示菜谱主图，带渐变遮罩显示基本信息
2. **基本信息** - 作者信息和营养信息
3. **所需材料** - 食材清单，支持勾选功能
4. **制作步骤** - 图文并茂的制作步骤
5. **底部操作栏** - 收藏、分享、开始制作按钮

### 🎯 主要功能

#### 1. 菜谱信息展示
- 菜谱标题、难度、时间、份量
- 评分和描述
- 作者信息
- 营养信息

#### 2. 材料清单
- 食材名称和用量
- 可勾选的购物清单功能
- 清晰的分类展示

#### 3. 制作步骤
- 分步骤图文说明
- 每个步骤的详细描述
- 制作小贴士
- 步骤图片展示

#### 4. 交互功能
- **收藏功能** - 本地存储收藏状态
- **分享功能** - 支持微信分享和复制链接
- **营养信息** - 点击查看详细营养数据
- **作者信息** - 点击查看作者主页

### 🔧 技术实现

#### 数据结构
```javascript
recipe: {
  id: 1,
  title: '宫保鸡丁',
  image: '主图URL',
  difficulty: '简单',
  time: '30分钟',
  servings: '2-3人份',
  rating: 4.8,
  author: '美食达人',
  description: '菜谱描述',
  ingredients: [
    { name: '鸡胸肉', amount: '300g', unit: '克' }
  ],
  steps: [
    {
      step: 1,
      title: '准备食材',
      description: '详细步骤说明',
      image: '步骤图片URL',
      tips: '制作小贴士'
    }
  ],
  nutrition: {
    calories: '320',
    protein: '28g',
    fat: '18g',
    carbs: '12g'
  }

```

#### 页面跳转
```javascript
// 从首页跳转到详情页
wx.navigateTo({
  url: `/pages/recipe-detail/recipe-detail?id=${recipeId}`
})
```

### 📝 使用说明

1. **页面访问** - 通过首页菜谱卡片点击进入
2. **收藏功能** - 点击底部收藏按钮，支持本地存储
3. **分享功能** - 点击分享按钮，支持微信分享和复制链接
4. **制作步骤** - 按照步骤图文说明进行制作
5. **材料清单** - 可勾选已准备的食材

### 🎨 设计特色

- **响应式布局** - 适配不同屏幕尺寸
- **渐变遮罩** - 主图区域美观的信息展示
- **步骤编号** - 清晰的制作步骤标识
- **交互反馈** - 按钮点击效果和状态变化
- **分享弹窗** - 优雅的分享选项展示
