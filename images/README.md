# Tab图标说明

由于无法直接创建图片文件，你需要手动添加以下图标到 `images/` 目录：

## 需要的图标文件

### 首页图标
- `home.png` - 首页未选中状态图标（建议尺寸：81x81px）
- `home-active.png` - 首页选中状态图标（建议尺寸：81x81px）

**图标建议：**
- 未选中：灰色炊具/菜盘图标
- 选中：绿色炊具/菜盘图标

### 分类图标
- `category.png` - 分类未选中状态图标（建议尺寸：81x81px）
- `category-active.png` - 分类选中状态图标（建议尺寸：81x81px）

**图标建议：**
- 未选中：灰色食材/列表图标
- 选中：绿色食材/列表图标

### 收藏图标
- `favorite.png` - 收藏未选中状态图标（建议尺寸：81x81px）
- `favorite-active.png` - 收藏选中状态图标（建议尺寸：81x81px）

**图标建议：**
- 未选中：灰色五角星图标
- 选中：绿色五角星图标

### 我的图标
- `profile.png` - 我的未选中状态图标（建议尺寸：81x81px）
- `profile-active.png` - 我的选中状态图标（建议尺寸：81x81px）

**图标建议：**
- 未选中：灰色用户图标
- 选中：绿色用户图标

## 图标获取方式

1. **设计工具**：使用 Figma、Sketch、Adobe XD 等设计工具制作
2. **图标库**：从 iconfont、iconpark 等图标库下载
3. **AI生成**：使用 AI 工具生成图标
4. **在线工具**：使用 Canva、稿定设计等在线设计工具

## 注意事项

- 图标尺寸建议为 81x81px
- 图标格式为 PNG，支持透明背景
- 未选中状态使用灰色系
- 选中状态使用绿色系（#3cc51f）
- 图标风格保持一致
